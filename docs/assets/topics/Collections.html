Why not Arrays?

  • Size is fixed (can't grow or shrink).
  • Arrays are useful when maximum number of objects can be guessed.
  • Let's see an example.

ArrayList

  • An ArrayList is an ordered list of reference type items, that comes with Java.
  • Each item in an ArrayList is known as an element.
  • The statement import java.util.ArrayList; enables use of an ArrayList.

How to create an ArrayList

  • We have defined and instantiated an ArrayList in the first line below.
  • We can also Add and Remove items to and from the ArrayList 

    	ArrayList students = new ArrayList();
	students.add(new Student("Alex"));
	students.add(new Student("Rose"));
	students.remove(0);

Generic ArrayList

  • We can restrict the type of the objects we hold in the ArrayList using < Type > 

ArrayList< Student > students = 
			new ArrayList< Student >();
students.add(new Student("Alex"));
students.add(new Student("Rose"));
students.remove(0);
students.remove(new Student("Alex"));
Student student = students.get(0);
System.out.println(student);

LinkedList

  • LinkedList and ArrayList are both subclass of List
  • ArrayList is implemented by an array
  • LinkedList is implemented by a doubly linked list
  • It is used like an ArrayList because they are siblings! (subclass of List)

Add/Remove Elements from LinkedList

Using LinkedList

  • Very similar to ArrayList:

List< String > list = new LinkedList();
list.add("Alex");
list.add("Rose");
System.out.println(list.get(0));
list.remove("Alex");
for (String string : list) {
	System.out.println(string);
}
